What for account at syncloud.it is needed?
Syncloud provides free second level domain name for each Syncloud device. This domain name is linked to you device address. Account at syncloud.it is needed to enable control over these domain names.
How can I access my Syncloud device?
Each Syncloud device has a domain name associated with it. You provide this domain name during device activation. For example if you provided "mydomain" then your device should be accessible at http://mydomain.syncloud.it address. You can always get device address after logging into syncloud.it.
What are login and password to access my device?
You specify these credentials at device activation along with device domain name. These credentials are not transfered anywhere, only you know them.
Can I access my device from anywhere through Internet?
Yes, you can. By default Syncloud device is in internal mode. That means it is not available outside of the home network. You can switch Syncloud device to external mode. This will make it available from anywhere through Internet as long as Syncloud device is connected to Internet.
Is it possible to use HTTPS with Syncloud device?
HTTPS can be enabled in the device settings panel. Right now Syncloud works with Let's Encrypt and self-signed certificates. All interactions with Let's Encrypt are automated. So you don't need to do anything to get truly secure connection.
Where Syncloud stores data?
Syncloud stores all data either on the device itself or on attached external hard drive. This includes all data of Syncloud applications. It's not necessary to get external hard disk to start using Syncloud. There are always few free gigabytes on SD card or flash memory, this space is used by Syncloud as a storage.
